# 🤖 Discord Bot v2

## 📁 الملفات
```
index.js      ← الكود الرئيسي
config.js     ← الإعدادات
package.json  ← المكتبات
data.json     ← يتولد تلقائي عند التشغيل
```

## ⚙️ طريقة التشغيل

### 1. ثبّت المكتبات
```bash
npm install
```

### 2. ضع التوكن في config.js
```js
token: "YOUR_BOT_TOKEN_HERE",
```
> الأفضل تستخدم `.env` ملف بدل ما تحط التوكن مباشرة

### 3. شغّل البوت
```bash
node index.js
# أو للتطوير
npm run dev
```

## 📋 الأوامر

| الأمر | الوصف |
|-------|--------|
| `+ping [voiceChannelId]` | يدخل/يطلع البوت من روم صوتي |
| `+status [voiceChannelId]` | يفعّل ستاتس الأعضاء |
| `+dashboard` | يفتح لوحة التحكم الرئيسية |

---

## 🎛️ Dashboard - الخيارات

### Setup New Ticket
- يسأل عن العنوان، الوصف، اسم الزر، إيموجي، صورة، روم البنل، رتبة الإدارة
- يرسل بنل تذاكر كامل

### Setup Auto Line
- يطلب صورة → يطلب رومات → كل من كتب في الرومات يرسله البوت الصورة تلقائي

### Setup Security
- حظر روابط
- حظر كلام وصخ
- حظر صور
- حظر سبام

### Setup Auto Response
- تضيف trigger ورد تلقائي
